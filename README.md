# 🏫 Sistem Informasi Madrasah Diniyah & PPDB Online

Sebuah platform Sistem Informasi Akademik dan Penerimaan Peserta Didik Baru (PPDB) terintegrasi untuk Madrasah Diniyah. Dibangun dengan fokus pada kecepatan, keamanan, dan antarmuka pengguna (UI/UX) yang modern & responsif.

## ✨ Fitur Utama

Sistem ini memiliki 3 hak akses (*Role*) utama:

* **👨‍💻 Admin / Kepala Madrasah**
    * Manajemen Data Master (Guru, Santri, Kelas, Mapel).
    * Verifikasi & Validasi pendaftaran PPDB.
    * Manajemen Konten Website (Berita, Agenda, Galeri, Profil).
    * Pengaturan Kontak & Informasi Madrasah.
* **👨‍🏫 Guru**
    * Dashboard jadwal mengajar terintegrasi.
    * Input absensi santri harian dengan rekapitulasi otomatis (Hadir, Sakit, Izin, Alpha).
    * Cetak Laporan Absensi Harian (Format Cetak Profesional).
* **👨‍👩‍👦 Wali Santri**
    * Pendaftaran akun mandiri.
    * Formulir Pendaftaran PPDB Online.
    * Cek status penerimaan secara *real-time*.
    * Melihat rekam jejak akademik (Data Santri) setelah resmi diterima.

## 🛠️ Teknologi yang Digunakan

* **Framework:** [Laravel](https://laravel.com/) (PHP)
* **Frontend:** [Livewire 3](https://livewire.laravel.com/) & [Alpine.js](https://alpinejs.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Database:** MySQL
* **Icons:** [Phosphor Icons](https://phosphoricons.com/)
